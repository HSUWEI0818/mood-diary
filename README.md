# mood-diary
A simple Mood Diary Web App  
Link: https://hsuwei0818.github.io/mood-diary/  
  
Prompt:  
You are Vibe Coding. Please build a simple “Mood Diary Web App”. Requirements: - The app allows users to write a daily diary entry. - The user can choose their mood (Happy, Sad, Angry, Neutral, Tired) using buttons or icons. - The diary entry + mood should be saved to localStorage. - The app should show the list of previously saved entries. - Clean UI, centered layout, pastel color theme. - No login system needed. - Deploy automatically and provide the public URL. - Provide the Git repository link.
